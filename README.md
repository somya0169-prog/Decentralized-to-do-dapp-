# Decentralized-to-do-dapp
Decentralized To-Do List DApp built with Python, Flask and Web3.py
# 🚀 Decentralized To-Do List DApp

## Features
- Add/Delete Tasks
- Timestamp Storage
- SHA-256 Hashing
- Dashboard Analytics
- Wallet Login (Simulated)
- Dark Mode Toggle
- Animated UI
- Mobile Responsive

## Tech Stack
- Python (Flask)
- HTML/CSS/JS
- Bootstrap

## Run
pip install flask
python app.py
